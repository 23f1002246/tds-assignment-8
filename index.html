Financial Metrics & Formulas

Return on Equity (ROE)

$$ROE = \frac{Net\ Income}{Shareholders'\ Equity} = \frac{485M}{2.1B} = 23.1\%$$

Earnings Per Share (EPS)

$$EPS = \frac{Net\ Income - Preferred\ Dividends}{Weighted\ Average\ Shares} = \frac{485M - 15M}{137.5M} = \$3.42$$

Price-to-Earnings Ratio

$$P/E\ Ratio = \frac{Market\ Price\ per\ Share}{Earnings\ per\ Share} = \frac{68.40}{3.42} = 20.0x$$

Revenue Analysis Pipeline

import pandas as pd
import numpy as np
from sklearn.linear_model import LinearRegression

# Load quarterly revenue data
revenue_data = pd.read_csv('quarterly_revenue.csv')
revenue_data['quarter'] = pd.to_datetime(revenue_data['quarter'])
revenue_data['revenue_millions'] = revenue_data['revenue'] / 1e6

# Calculate year-over-year growth
revenue_data['yoy_growth'] = revenue_data.groupby(
    revenue_data['quarter'].dt.quarter
)['revenue_millions'].pct_change(periods=4) * 100

# Forecast next quarter using linear regression
model = LinearRegression()
X = np.arange(len(revenue_data)).reshape(-1, 1)
y = revenue_data['revenue_millions'].values

model.fit(X, y)
next_quarter_forecast = model.predict([[len(revenue_data)]])[0]

print(f"Q1 2025 Revenue Forecast: ${next_quarter_forecast:.1f}M")
print(f"Growth Rate: {revenue_data['yoy_growth'].iloc[-1]:.1f}%")

Output: Q1 2025 Revenue Forecast: $625.3M | Growth Rate: 18.2%

Customer Segmentation Query

-- Customer value analysis by segment
SELECT 
    customer_segment,
    COUNT(DISTINCT customer_id) as customer_count,
    AVG(account_balance) as avg_balance,
    SUM(quarterly_revenue) as total_revenue,
    AVG(CASE 
        WHEN churn_risk = 'HIGH' THEN 1 
        ELSE 0 
    END) as churn_rate,
    RANK() OVER (
        ORDER BY SUM(quarterly_revenue) DESC
    ) as revenue_rank
FROM customer_analytics
WHERE quarter = 'Q4_2024'
GROUP BY customer_segment
HAVING COUNT(DISTINCT customer_id) > 100
ORDER BY total_revenue DESC;
